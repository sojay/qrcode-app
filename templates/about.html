how we built?

I worked on a QR code generating SaaS (software as a service) website. With only a few lines of Python, we built a QR generator from scratch. I registered the domain QRCodeMaker.ca which should be up and running tomorrow, giving billions of people around the world a quick and easy way to generate QR codes.

This project takes some very simple Python code to translate a string into a QR code, and build a website around it in flask, turning our very simple program into a very powerful resource. There are a number of things that we could do to expand this project such as data validation, better logging, and an internal dashboard.
